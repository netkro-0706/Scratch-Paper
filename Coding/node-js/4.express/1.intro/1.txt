# Express
http모듈로 웹 서버를 만들 때 코드가 보기 좋지 않고 확장성도 떨어짐
이것을 프레임워크로 해결할 수 있는데 대표적인게 Express, Koa, Hapi등이 있다.

app.set("port", 포트) 로 서버가 실행 될 포트 지정
app.get("주소", 라우터)로 GET 요청이 올 때 어떤 동작을 할지 지정
app.listen("포트", 콜백)으로 몇 번 포트에서 서버를 실행할지 지정

## nodemon
패키지중 하나로 코드를 수정하면 자동으로 갱신하여 서버를 재실행해주는 자동 재시작 도구이다.
개발할 때에는 node app.js보다 nodemon app.js으로 자주 사용한다.
html은 감시하지 않는다. html파일을 수정했다면 새로고침을 해야한다.
