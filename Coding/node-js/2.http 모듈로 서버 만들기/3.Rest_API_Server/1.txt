Rest API
서버에 요청을 보낼 때는 주소를 통해서 요청의 내용을 표현한다.
1./index.html이면 index.html을 보내달라는 뜻이다.
2.서버가 이해하기 좋은 주소가 좋다.

REST API - Representational State Transfer
서버의 자원을 정의하고 자원에 대한 주소를 지정하는 방법

HTTP 요청 메서드
GET : 서버 자원을 가져오려고 할 때 사용
POST : 서버에 자원을 새로 등록하고자 할 때 사용(무얼 사용하면 되는지 애매할 때)
PUT : 서버의 자원을 요청에 들어있는 자원으로 치환하고자 할 때 사용(전체 수정)
PATCH : 서버 자원의 일부만 수정하고자 할 때 사용(일부 수정)
DELETE : 서버 자원을 삭제하고자 할 때 사용
